## Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/niteshpk/postboy.git
   cd postboy
   ```
2. Install dependencies:
   ```sh
   npm install
   ```
3. Start the development server:
   ```sh
   npm start
   ```
4. Build for production:
   ```sh
   npm run build
   ```
